# Quiz PE 12 - 03.05.2026

## Textaufgabe - 20P

```
Du möchtest ein Programm schreiben, das den Benutzer mindestens einmal nach einer Zahl fragt.
Das Programm soll so lange weiterfragen, bis der Benutzer 0 eingibt.

Welche Python-Struktur eignet sich dafür am besten?

Lösung:
```
```python
while True
    try
        zahl = int(input("Bitte geben Sie eine Zahl ein: "))
    except
        print("Ungültige Eingabe")
    else
        if zahl == 0
            break
```
