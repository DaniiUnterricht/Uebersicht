# Visitenkarte

## Lernziel:
- Ausgabe
- Variablen
- Rechnen

## Aufgabe:

- Programmieren Sie eine Visitenkarte, die Vorname, Nachname und Alter ausgibt.
- Danach eine weitere Visitenkarte mit den selben Eigenschaften.
- Und zum Schluss eine Ausgabe des Altersunterschied von der ersten zu zweiten Person.

## Ausgabe:

```
Person 1:
Bitte geben Sie den Vornamen ein: Max
Bitte geben Sie den Nachnamen ein: Mustermann
Bitte geben Sie das Alter an: 30

Person 2:
Bitte geben Sie den Vornamen ein: Dani
Bitte geben Sie den Nachnamen ein: Wittmeier
Bitte geben Sie das Alter an: 26

Ausgabe:
Max Mustermann ist 30 Jahre alt und hat zu Dani Wittmeier ( 26 ) einen Altersunterschied von 4 Jahren.
```

## Tipp

Um eine Zahl defintiv Positiv haben zu können gibt es den Befehl: Math.Abs(variable)

Bsp. variable = Math.Abs(variable);
