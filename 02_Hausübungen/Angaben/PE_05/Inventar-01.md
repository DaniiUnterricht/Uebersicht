# Inventar-01

## Lernziel:

- Arrays
- Strings
- For-Schleifen
- Spiel-Logik

## Aufgabe:

Ein Spieler besitzt ein **Inventar mit 4 Gegenständen**.

Der Benutzer soll **4 Gegenstände eingeben**, die anschließend im Inventar gespeichert werden.

Danach sollen alle Gegenstände ausgegeben werden.

## Ausgabe:

```
Inventar
========

Gegenstand 1 eingeben: Schwert
Gegenstand 2 eingeben: Schild
Gegenstand 3 eingeben: Trank
Gegenstand 4 eingeben: Schlüssel

Inventar:
Schwert
Schild
Trank
Schlüssel
```

## Tipp:

- Verwenden Sie ein **String-Array**.
- Verwenden Sie eine **for-Schleife**, um die Gegenstände auszugeben.
