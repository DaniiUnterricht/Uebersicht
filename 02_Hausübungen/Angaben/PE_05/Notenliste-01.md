# Notenliste-01

## Lernziel:

- Arrays
- For-Schleifen
- Berechnungen
- Durchschnitt berechnen
- Logisches Denken

## Aufgabe:

Programmieren Sie ein Programm, das **5 Noten** in ein Array einliest.

Anschließend soll das Programm:

- alle Noten ausgeben
- den **Notendurchschnitt** berechnen
- die **beste Note**
- und die **schlechteste Note** ausgeben

## Ausgabe:

```
Notenliste
==========

Note 1 eingeben: 2
Note 2 eingeben: 1
Note 3 eingeben: 3
Note 4 eingeben: 2
Note 5 eingeben: 4

Noten:
2
1
3
2
4

Durchschnitt: 2,4
Beste Note: 1
Schlechteste Note: 4
```

## Tipp:

- Speichern Sie die Noten in einem **int-Array**.
- Für den Durchschnitt benötigen Sie eine **Summe**.
- Die beste und schlechteste Note können mit Vergleichsoperatoren gefunden werden.
