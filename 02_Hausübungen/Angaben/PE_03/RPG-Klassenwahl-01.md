# RPG-Klassenwahl-01

## Lernziel:

- Switch Statement

- Mehrere Variablen verwalten

- Rechenoperationen

- Strukturierte Konsolenausgabe

## Aufgabe:

Programmieren Sie ein kleines RPG-Startsystem.

Der Spieler wählt eine Klasse:
```
1 - Krieger
2 - Magier
3 - Bogenschütze
4 - Heiler
```

Startwerte:

- Krieger → HP 120, Mana 20, Angriff 15

- Magier → HP 70, Mana 120, Angriff 20

- Bogenschütze → HP 90, Mana 50, Angriff 18

- Heiler → HP 80, Mana 100, Angriff 10

Danach wählt der Spieler eine Waffe:
```
1 - Schwert (+5 Angriff)
2 - Stab (+10 Mana)
3 - Bogen (+7 Angriff)
```
Die Werte sollen angepasst und anschließend vollständig ausgegeben werden.

## Ausgabe:

```
RPG-Klassenauswahl
==================

1 - Krieger
2 - Magier
3 - Bogenschütze
4 - Heiler

Wähle deine Klasse (1-4): 2

1 - Schwert (+5 Angriff)
2 - Stab (+10 Mana)
3 - Bogen (+7 Angriff)

Wähle deine Waffe (1-3): 2

Deine Klasse: Magier
HP: 70
Mana: 130
Angriff: 20
```

## Tipp:
- Verwenden Sie mehrere Variablen (HP, Mana, Angriff).

- Nutzen Sie zwei Switch-Blöcke.

- Achten Sie auf eine saubere Struktur.
