# Rabatt-System

## Lernziel
- Reihenfolge von IF Systemen

## Aufgabe
- Programmieren Sie ein System, welches Berechnet welcher Rabatt gewährt wird:
1. User gibt Einkaufswert ein.
2. Rabatt wird berechnet und Ausgegeben
    - Ab 100 Euro -> 10%
    - Ab 250 Euro -> 20%
    - Ab 500 Euro -> 30%

## Ausgabe
```
Rabatt-System
=============

Bitte geben Sie Ihren Einkaufswert ein: 300

Ihr Rabatt beträgt 20%!

## Tipp
