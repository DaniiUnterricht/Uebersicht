# Parkticket-Automat

## Lernziel
- Bereichsprüfung
- Logische Kombination

## Aufgabe

Der Nutzer soll die Parkdauer eingeben und ob es Wochenende ist ( true / false )

Preise:
- 1 bis 2 Stunden -> 2 Euro
- 3 bis 5 Stunden -> 5 Euro
- Mehr als 5 Stunden -> 10 Euro
- Am Wochenende -> +50% Zuschlag

## Ausgabe
```
Parkticket Automat
==================

Bitte geben Sie Parkdauer in Stunden an: 3
Ist es am Wochenende ( t / f ): t

Sie müssen 7,50 Euro bezahlen.
```

## Tipp
- Speichern Sie ob es am Wochenende ist in eine boolische Variable.
- Berechnen Sie den 50% Zuschlag in einer eigenen IF Bedingung.
