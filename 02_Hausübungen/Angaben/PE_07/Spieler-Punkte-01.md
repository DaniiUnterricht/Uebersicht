# Spieler-Punkte-01

## Lernziel:

* Jagged Arrays
* Methoden
* Summen berechnen

## Aufgabe:

Mehrere Spieler haben unterschiedlich viele Runden gespielt.

Die Punkte werden in einem Jagged Array gespeichert.

Diesen Jagged Array dürfen Sie selbst befüllen.

Programmieren Sie ein Programm, das:

* ein Jagged Array mit Punkten erstellt
* für jeden Spieler die **Gesamtpunkte berechnet**

---

👉 Erstellen Sie dazu eine Methode:

* erhält ein Array (Punkte eines Spielers)
* berechnet die Summe
* gibt die Summe zurück

---

Anschließend soll ausgegeben werden:

* „Spieler X hat Y Punkte erreicht“

## Ausgabe:

```
Spieler-Punkte-01
=================

Spieler 1 hat 30 Punkte erreicht
Spieler 2 hat 15 Punkte erreicht
Spieler 3 hat 24 Punkte erreicht
```

## Tipp:

* Schleife innerhalb der Methode
* Methode mehrfach aufrufen
