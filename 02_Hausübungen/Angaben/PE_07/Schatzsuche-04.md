# Schatzsuche-04

## Lernziel:

* Jagged Arrays
* Methoden kombinieren
* Logik & Zufall

## Aufgabe:

Ein Dungeon besteht aus unterschiedlich langen Reihen.

Jede Position enthält:

* `0` = leer
* `1` = Schatz

Programmieren Sie ein Programm, das:

* ein Jagged Array mit Zufallswerten erstellt
* Mit 2 - 4 Reihen
* jede Reihe unterschiedlich lang macht ( 2 - 4 )

👉 Erstellen Sie zwei Methoden:

1. Methode:

   * erstellt eine Reihe mit Zufallswerten
   * gibt diese zurück

2. Methode:

   * zählt alle Schätze im gesamten Dungeon
   * gibt die Anzahl zurück

Am Ende soll ausgegeben werden:

* Gesamtanzahl der Schätze
* gesamte Dungeon-Struktur

## Ausgabe:

```
Schatzsuche-04
=====================

Schätze insgesamt: 5

Dungeon:
1 0 1
0 0
1 1 0 1
```

## Tipp:

* Random verwenden
* Methoden sauber trennen
* Jagged Array flexibel aufbauen
