# Schaden-Protokoll-01

## Lernziel:

- Zweidimensionale Arrays  
- For-Schleifen  
- Benutzereingaben  
- Berechnungen  

## Aufgabe:

Ein Spieler verursacht Schaden in mehreren Runden.

Der Benutzer gibt die Anzahl an Level vor

Der Benutzer soll die Werte für ein **Xx3 Array** selbst eingeben.

Anschließend soll das Programm:

- die **Summe jeder Zeile einzeln** berechnen  
- zusätzlich die **Gesamtsumme** ausgeben  

## Ausgabe:

```
Schaden-Protokoll-01
====================

Anzahl Level: 2

Level 1
Runde 1: 2
Runde 2: 3
Runde 3: 1

Level 2
Runde 1: 10
Runde 2: 2
Runde 3: 3

Schaden Level 1: 6
Schaden Level 2: 15

Gesamtschaden: 21
```

## Tipp:

- Nutzen Sie eine zweite Schleife für die Berechnung  
- Setzen Sie die Summe pro Zeile zurück, verwenden Sie eine Variable für die Gesamtsumme
