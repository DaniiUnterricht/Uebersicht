# Summe-01

## Lernziel:

- Do-While-Schleife
- Benutzerinteraktion
- Variablen
- Addition

## Aufgabe:

Der Benutzer soll wiederholt **Zahlen eingeben**.

Alle Zahlen sollen **addiert werden**.

Die Eingabe endet, wenn der Benutzer **0** eingibt.

Am Ende soll die **Gesamtsumme** ausgegeben werden.

## Ausgabe:

```
Summe
=====

Bitte Zahl eingeben (0 beendet): 5
Bitte Zahl eingeben (0 beendet): 7
Bitte Zahl eingeben (0 beendet): 3
Bitte Zahl eingeben (0 beendet): 0

Summe: 15
```

## Tipp:

- Verwenden Sie eine **do-while-Schleife**, damit mindestens eine Zahl eingegeben wird.
