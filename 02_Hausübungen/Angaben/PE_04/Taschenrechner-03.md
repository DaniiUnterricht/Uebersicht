# Taschenrechner-03

## Lernziel:

- While-Schleife
- Do-While-Schleife
- Switch Statement
- Benutzerinteraktion
- Grundrechenarten

## Aufgabe:

Programmieren Sie einen **einfachen Taschenrechner**.

Der Benutzer kann eine Rechenoperation auswählen:

```
1 - Addition
2 - Subtraktion
3 - Multiplikation
4 - Division
0 - Beenden
```

Nach der Auswahl soll der Benutzer **zwei Zahlen eingeben**.

Das Programm soll das Ergebnis berechnen und ausgeben.

Anschließend soll das Menü **erneut angezeigt werden**.

Das Programm läuft so lange, bis der Benutzer **0 eingibt**.

## Ausgabe:

```
Taschenrechner
==============

1 - Addition
2 - Subtraktion
3 - Multiplikation
4 - Division
0 - Beenden

Bitte wählen Sie eine Operation: 1

Zahl 1: 5
Zahl 2: 3

Ergebnis: 8

---

Bitte wählen Sie eine Operation: 3

Zahl 1: 8
Zahl 2: 5

Ergebnis: 40

---

Bitte wählen Sie eine Operation: 0

```

## Tipp:

- Verwenden Sie eine **Schleife**, damit das Programm wiederholt ausgeführt wird.
- Verwenden Sie ein **switch-Statement** für die Auswahl der Rechenoperation.
