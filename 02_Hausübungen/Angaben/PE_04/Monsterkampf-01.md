# Monsterkampf-01

## Lernziel:

- While-Schleife
- Variablen
- Benutzerinteraktion
- Grundrechnen
- Spiel-Logik

## Aufgabe:

Programmieren Sie einen **einfachen Monsterkampf**.

Der Spieler kämpft gegen ein Monster.

Zu Beginn hat das Monster **20 Lebenspunkte**.

Der Spieler greift das Monster in jeder Runde an und gibt den **Schaden** ein.

Der Schaden wird von den Lebenspunkten des Monsters abgezogen.

Das Programm soll den Kampf so lange wiederholen, bis das Monster **0 oder weniger Lebenspunkte** hat.

Am Ende soll ausgegeben werden:

```
Monster besiegt!
```

## Ausgabe:

```
Monsterkampf
============

Monster-Leben: 20
Schaden eingeben: 5

Monster-Leben: 15
Schaden eingeben: 7

Monster-Leben: 8
Schaden eingeben: 9

Monster besiegt!
```

## Tipp:

- Verwenden Sie eine **while-Schleife**, solange das Monster noch Lebenspunkte hat.
- Ziehen Sie den Schaden von den Lebenspunkten ab.
