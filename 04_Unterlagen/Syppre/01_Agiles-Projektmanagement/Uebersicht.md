# 🚀 Agiles Projektmanagement – Übersicht

---

# 🧠 Was ist agiles Projektmanagement?

Agiles Projektmanagement bedeutet:

- flexibel arbeiten  
- schnell auf Änderungen reagieren  
- in kleinen Schritten entwickeln  
- regelmäßig verbessern  

👉 Fokus:
**funktionierende Software statt perfekte Planung**

---

# 🔁 Grundprinzipien

- Iteratives Arbeiten (in Schleifen)
- enge Zusammenarbeit im Team
- Feedback nutzen
- kontinuierliche Verbesserung

---

# 📦 Wichtige Methoden

---

# 🧩 Scrum

## 🧠 Beschreibung
Arbeiten in festen Zeitabschnitten (**Sprints**)

## ⚙️ Merkmale
- Product Backlog
- Sprint Backlog
- feste Rollen (Product Owner, Scrum Master, Team)
- regelmäßige Meetings

## 🎯 Fokus
Strukturierte, iterative Entwicklung

---

# 📋 Kanban

## 🧠 Beschreibung
Visualisierung von Aufgaben auf einem Board

## ⚙️ Merkmale
- kein Sprint
- kontinuierlicher Workflow
- Status-Spalten (z. B. To Do → Doing → Done)

## 🎯 Fokus
Konstanter Arbeitsfluss

---

# 🚀 Extreme Programming (XP)

## 🧠 Beschreibung
Fokus auf hohe Code-Qualität

## ⚙️ Merkmale
- Pair Programming
- Test Driven Development (TDD)
- kleine Releases

## 🎯 Fokus
Sauberer, stabiler Code

---

# 🏭 Lean Software Development

## 🧠 Beschreibung
Effiziente Entwicklung ohne Verschwendung

## ⚙️ Prinzipien
- nur das Nötige bauen
- schnelle Lieferung
- einfache Lösungen

## 🎯 Fokus
Effizienz und Geschwindigkeit

---

# 🔄 Crystal

## 🧠 Beschreibung
Anpassbare Methode je nach Projekt

## ⚙️ Merkmale
- keine festen Regeln
- abhängig von Teamgröße und Projekt

## 🎯 Fokus
Flexibilität

---

# 🔀 Feature-Driven Development (FDD)

## 🧠 Beschreibung
Entwicklung basiert auf Features

## ⚙️ Ablauf
- Feature planen
- Feature entwickeln
- Feature fertigstellen

## 🎯 Fokus
Klare Struktur nach Funktionen

---

# ⚡ Agile ohne festes Framework

## 🧠 Beschreibung
Viele Teams arbeiten agil ohne feste Methode

## ⚙️ Merkmale
- eigene Prozesse
- flexible Planung
- regelmäßige Abstimmung

## 🎯 Fokus
Praktische Umsetzung statt strikter Regeln

---

# 🧠 Vergleich

| Methode | Besonderheit |
|--------|-------------|
| Scrum | Arbeiten in Sprints |
| Kanban | kontinuierlicher Workflow |
| XP | Fokus auf Code-Qualität |
| Lean | Effizienz |
| Crystal | flexibel |
| FDD | Feature-basiert |

---

# 🏁 Zusammenfassung

Agiles Projektmanagement bedeutet:

- flexibel planen  
- in kleinen Schritten arbeiten  
- regelmäßig verbessern  
- auf Änderungen reagieren  

👉 Ziel:
**Schnell funktionierende Software liefern**
